# Post queue

Drop your Mavrx photos and videos here. The bot posts **one per day** (newest/best first) and won't repeat one it already posted.

- Images: `.jpg` `.png` `.webp` — Videos: `.mp4` `.mov` `.m4v`
- Add files by dragging them in, or from your phone with the **GitHub mobile app** (Add file → Upload files), then commit.
- When this folder runs out of unused files, you'll get a Telegram message saying the queue is empty — just upload more.
- Keep videos reasonably small (under ~50 MB each).
